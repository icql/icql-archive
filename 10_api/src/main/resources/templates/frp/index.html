Frp内网穿透

使用说明：
        1）下载frpc客户端，配置frpc.ini，然后双击启动run.bat即可。也可以nssm将此应用注册为windows服务，开机自启后台运行，避免每次需要手动开启。
        2）管理界面

注意：
        1）frpc.ini配置文件中端口范围取 57003-57010 中的一个，若全部被占用，请邮件联系 chenqingin@icql.work
        2）使用nssm安装win服务时，先在cmd命令窗口中输入nssm install 服务名称，跳出安装窗口，启动应用选择 frpc.exe，参数填入：frpc.exe -c frpc.ini，这里的frpc.exe和frpc.ini均需要完整路径，且设置完成后不能移动文件夹
        3）若使用的是转发远程桌面，则双击run.bat或者注册win服务成功后，可以在别的电脑mstsc远程此电脑，地址为 api.icql.work:你设置的端口
