ICQL Bookmark Sync

注册地址：点击
使用说明：
        点击注册后请及时登录注册邮箱，点击激活链接，然后请在chrome系浏览器输入网址 chrome://extensions 打开，拖拽下载的扩展文件（.crx文件）到浏览器的扩展窗口点击添加（新版浏览器谷歌对第三方插件进行了限制，请解压.crx文件， 浏览器扩展窗口选择开发者模式——加载已解压的扩展程序——选择解压的文件夹），再点击浏览器右上角"书本标志"的扩展， 登录账户并设置同步时间间隔，至此配置完成。

提供接口：
/bookmark/data?name=xxx&version=xxx 提供json数据，version为空时取最新版
/bookmark/data/page?name=xxx&version=xxx 提供简单页面展示
